# Este proyecto consiste en el desarrollo de un MicroSaaS, una aplicación web funcional diseñada para gestionar las operaciones diarias de un pequeño negocio mediante un sistema CRUD completo de cuatro entidades: usuarios, clientes, productos y pedidos. Para hacerlo posible, he aplicado diversas soluciones tecnológicas, comenzando por el diseño del frontend. Utilizando HTML5, he estructurado la semántica de los documentos web, creando formularios y tablas de datos accesibles. Además, con CSS3 y el framework Bootstrap, he implementado una interfaz de usuario moderna e intuitiva, aplicando un diseño personalizado estilo "glassmorphism" que facilita la navegación y mejora la experiencia visual.
A nivel lógico y de datos, la aplicación utiliza Python junto con el framework Flask como servidor backend, lo que me ha permitido manipular dinámicamente los documentos web y conectar la interfaz con una base de datos relacional SQLite. A través de consultas SQL, el sistema gestiona, importa y extrae la información de forma persistente. Toda esta arquitectura técnica se traduce en una solución directa para el usuario final: digitalizar y centralizar la gestión de su negocio. En lugar de depender de procesos manuales en papel o en hojas de cálculo desorganizadas, cualquier pequeño comercio o autónomo puede ahora llevar el control de su inventario y de las ventas a clientes de forma automatizada, reduciendo errores y ahorrando un tiempo valioso en su día a día.
En conclusión, el desarrollo de este proyecto me ha permitido integrar de forma práctica todos los conceptos vistos durante el curso, demostrando cómo la combinación estructurada de lenguajes de marcas, lenguajes de script de servidor y bases de datos resulta en una herramienta empresarial robusta y útil para la sociedad. Ha sido un reto conectar todas estas capas desde cero, pero el resultado es una aplicación escalable que cumple con los estándares reales del desarrollo web.
